Core Concept

What is Consensus?

Consensus is the mechanism by which distributed nodes agree on the canonical state of the blockchain without a central authority.

📋 Definition

A consensus algorithm is a fault-tolerant mechanism used in distributed computing and multi-agent systems to achieve the necessary agreement on a single data value or a single state of the network among distributed processes or multi-agent systems — such as with cryptocurrencies.

🛡️ Byzantine Generals Problem

Proposed by Lamport, Shostak & Pease (1982): several Byzantine army generals must coordinate an attack, but some generals may be traitors sending conflicting orders. How can loyal generals guarantee they all attack simultaneously despite traitors?

🔑 The Mapping:
Generals → Network nodes
Messages → Transactions & block proposals
Traitors → Malicious or faulty nodes
Attack Plan → Agreed blockchain state
💡 Solution:
BFT consensus requires that fewer than 1/3 of nodes are Byzantine. PoW solves this through economic incentives (Nakamoto Consensus). PBFT solves it through cryptographic voting rounds.
